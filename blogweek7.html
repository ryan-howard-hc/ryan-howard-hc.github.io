Week 7 - CLI

What have you learned about CLIs? Do you think they are scary?

At first glance, yes, CLI's are scary. After learning rudimentary coding experience, it's more of an experiential problem. What are they useful for? They allow for relatively easy for navigating through file systems through basic commands. They also provide built in systems that provide commands for this navigation and for commands to control the system. Another very important use for CLI's are that they allow scripting and automating repetitive tasks involved in command executions. Lastly, they allow for remote access/networking. They can be used to establish secure connections remotely and checking connectivity.

If you were to build a CLI, what would it do?

It could allow for importing, exporting files for backup or automated reading purposes. It could provide integration between an OS and programs or communication between other programs. It could allow for automated updating tasks or prioritizing tasks. Also could allow for customization in this same respect.
